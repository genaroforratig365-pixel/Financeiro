# Comentários sobre o front-end - V10

## 2025-11-07
- página inicial passou a exibir saudação, busca e seleção de usuários cadastrados antes de entrar no sistema; a escolha sincroniza a sessão local com nome/e-mail existentes e permite seguir direto para Saldo Diário.
- tela de "Dados do Usuário" detecta o acesso do Genaro e libera painel para cadastrar novos usuários, listando os identificadores já registrados e garantindo feedback específico.
- movimentação diária (`/saldo-diario`) ganhou formulários rápidos para registrar pagamentos, receitas, débitos por banco e atualização de saldo no próprio painel, com resumos por grupo e validações de valor/opções.
